# X Always Following

Skip the algorithm. See tweets from people you actually follow.

## What it does

Automatically switches your X.com timeline from "For you" to "Following" every time you open the site.

## Install

1. Clone this repo
2. Go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked**
5. Select this folder

## Why

Because your feed should be yours.
